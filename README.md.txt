# E-commerce Sales Analysis Dashboard

Analyze sales trends, top-performing products, and customer segments with an interactive dashboard.

**Dataset:** [Kaggle E-Commerce Data](https://www.kaggle.com/datasets/carrie1/ecommerce-data)

# To Run This Project in Your Local install python and its dependencies

Please install python 
python -m venv venv
source venv/bin/activate    # On Windows use: venv\Scripts\activate
pip install --upgrade pip
pip install -r requirements.txt
